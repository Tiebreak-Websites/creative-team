# SpaceX IPO Landing Page

One-page landing about the SpaceX IPO. Hero is a rotating moon with bright/dark sides — tap the toggle to flip between positive factors and risks.

## Stack
- HTML + Tailwind (CDN, no build step)
- GSAP + ScrollTrigger for animations
- Vanilla JS for moon toggle + starfield canvas

## Files
- `index.html` — markup + copy
- `styles.css` — moon stage, cards, sections
- `script.js` — starfield, hero entrance, scroll reveals, moon flip

## Run locally
Just open `index.html` in a browser. No install needed.

For a proper local server (recommended for testing):
```
npx serve .
```
